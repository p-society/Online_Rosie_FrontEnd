Welcome to IIIT Rosei

This app is made by the contributors at P-society and we request you to kindly follow the guidelines listed below.

This site is hosted on glitch and being a free service, it takes about 1 minute to wake the application up if there is no traffic on the server.

A lot of effort has been put into making this application, so please don't register with an ID that isn't your college ID.

If your account is not activated by midnight, irrespective of your registration, then it will automatically be deleted. This step is being taken to prevent any tampering with the database.

Despite non availability of funds, the app is being hosted on a server and is using premium email service to provide a better user experience, in case you forget your password or do not recieve an activation email.

Please use the application responsibly. In case of any inconvenience, you can open an issue at our P-society Github repository. (Kindly refrain from raising minor issues like slow loading for the first time)
